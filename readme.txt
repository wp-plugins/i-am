== Description ==
<h3>Become known to all.</h3>
'I am' is a widget plugin that shows who you are, ie you place the widget in the desired position, once configured in your admin panel.
You can customize the following options:
<ul>
  <li>Name</li>
  <li>Color Name</li>
  <li>Description</li>
  <li>Link</li>
  <li>Twitter url</li>
  <li>Facebook url</li>
  <li>Google url</li>
  <li>Header background</li>
  <li>Background Color</li>
  <li>And more ...</li>
</ul>
Prove to your followers who you are, a cool and modern way.
 
== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'I am'
3. Activate 'I am' from your Plugins page.
4. Visit 'Setting > 'I am'.
5. Visit 'Appearance > Widget -> 'I am'.
